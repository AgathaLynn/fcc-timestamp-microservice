A Timestamp Microservice

A simple API project built while working through FreeCodeCamp's curriculum. If you pass a date - either a unix timestamp or a natural-language date - as a parameter (in the url), the site will return both the unix and natural-language forms of the date.

Here are a couple of examples:

  • The url http://fcc-timestamp-microservice-agathalynn.c9users.io/4/3/2015 returns {"unix":1428019200,"natural":"April 3, 2015"}.
  • The url http://fcc-timestamp-microservice-agathalynn.c9users.io/-15062875200 returns {"unix":-15062875200,"natural":"September 4, 1492"}
  • The url http://fcc-timestamp-microservice-agathalynn.c9users.io/tickle-me-elmo returns {"unix":null,"natural":null}
